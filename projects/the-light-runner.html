THE LIGHT RUNNER

TRY IT OUT

You're a guy flying in a spaceship through an endless pipe, while avoiding obstacles. There are 8 different scenes with different music audios so you never get bored The game was built with the Unity 3D engine.

This game was made thanks to Jasper Filck's amazing Swirly Pipe tutorial. I went through it and then proceeded to change it so that it fitted my interests.

It's available for Desktop, Android and the Web.

This project was done from 2016 to 2017 (Watch trailer).

What did I gain from it?

  • Created a game by modifying a Unity 3D game tutorial (adapted to my needs and interests).
  • Released the game to the Google Play App Store.
  • Brand name and logo development -- JustHavFun

The skills I acquired doing this project include

  • Use of the game development tool Unity 3D.
  • Basics of C# programming language.
  • Managing Social media accounts.
